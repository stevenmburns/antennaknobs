1"""Log-periodic dipole array (LPDA) (L. B. Cebik, W4RNL; ARRL Antenna Book 

2LPDA chapter). 

3 

4A frequency-INDEPENDENT array: a row of parallel dipoles whose lengths and 

5spacings shrink geometrically toward the front by a constant ratio tau, fed 

6by a TRANSPOSED (phase-alternating) feeder boom. At any frequency in the 

7design band only the two or three dipoles near a half-wavelength form the 

8"active region" and radiate as a small driver/reflector/director group; the 

9active region slides along the array with frequency, so gain, pattern, and 

10feed impedance stay roughly constant over a very wide band. The beam fires 

11toward the SHORT (apex) end. This is a completely different design law from 

12every resonant antenna in the catalog. 

13 

14Two constants describe the array (with the band edges): 

15 tau = l(n+1)/l(n) = d(n+1)/d(n) element scaling ratio (< 1) 

16 sigma = d(n) / (2 * l(n)) relative spacing constant 

17with the optimum sigma_opt = 0.243*tau - 0.051. The half apex angle follows 

18from cot(alpha) = 4*sigma/(1 - tau). 

19 

20The transposed feeder is modeled with ideal transmission lines joining 

21adjacent element centres, every one CROSSED (the TL `transposed` flag -- a 

22port-B polarity inversion) to supply the 180-degree section-to-section 

23phase reversal. The array is driven at the front (shortest) element. 

24 

25CAVEAT -- feedpoint impedance: the feeder is modeled as a cascade of IDEAL, 

26LOSSLESS crossed transmission lines. That cascade has internal resonances 

27that NEC2's tl_card does not damp, so the computed driving-point impedance is 

28unreliable (it can swing wildly and even show a negative real part at 

29scattered in-band frequencies). A real LPDA's feeder loss plus a termination 

30stub behind the longest element suppress this. The robust, physically 

31meaningful outputs of this model are the GAIN and the forward PATTERN, which 

32stay LPDA-like (~6-9 dBi, fires toward the apex) across the whole band; treat 

33the SWR/impedance readout as indicative only. 

34 

35Geometry, in the framework's (x, y, z) convention: 

36 - x : boom axis; longest element at the back (x=0), shortest at the 

37 front (max x); the beam fires toward +x 

38 - y : the dipole length axis (all elements horizontal, centre at y=0) 

39 - z : constant height `base` 

40Horizontally polarised. 

41 

42 d0 d1 d2 d3 ... (lengths shrink by tau) 

43 | | | | <- crossed feeder -> F (driven, front) 

44 back (low freq) front (high freq), beam --> +x 

45""" 

46 

47from antennaknobs import AntennaBuilder 

48from antennaknobs.network import Driven, Network, PortOnWire, TL 

49from types import MappingProxyType 

50 

51 

52class Builder(AntennaBuilder): 

53 default_params = MappingProxyType( 

54 { 

55 # Operating/test frequency. The band runs from `freq_low_factor` 

56 # * design_freq upward; design_freq sits inside the active region. 

57 "design_freq": 28.57, 

58 "freq": 28.57, 

59 "base": 10.0, 

60 # Highest band edge as a fraction of design_freq: the shortest 

61 # (front) element is a half wave here. >1 so design_freq sits 

62 # just inside the top of the band, at the front of the active 

63 # region where the feed is -- the longer elements extend the 

64 # band downward. 

65 "freq_high_factor": 1.15, 

66 # Element scaling ratio and relative spacing constant. sigma near 

67 # the optimum sigma_opt = 0.243*tau - 0.051 (~0.168 for tau=0.9) 

68 # for good gain; lower sigma compresses the array and drops gain. 

69 "tau": 0.9, 

70 "sigma": 0.14, 

71 # Number of dipole elements. 

72 "n_elements": 10, 

73 # Feeder (boom) characteristic impedance in ohms; modeled crossed. 

74 "z0": 100.0, 

75 # Overall length trim of the active region. 

76 "length_factor": 1.0, 

77 "ui_params": MappingProxyType( 

78 { 

79 "target_z0": 50.0, 

80 "default_view": "xy", 

81 "n_elements": {"min": 4, "max": 16, "step": 1}, 

82 "tau": {"min": 0.8, "max": 0.95}, 

83 "sigma": {"min": 0.03, "max": 0.18}, 

84 "length_factor": { 

85 "min": 0.85, 

86 "max": 1.05, 

87 }, 

88 } 

89 ), 

90 } 

91 ) 

92 

93 def _layout(self): 

94 """Element half-lengths h[n] and boom positions x[n], longest (n=0) 

95 at the back. Shared by build_wires and build_network.""" 

96 c = 299.792458 

97 n = int(self.n_elements) 

98 tau = self.tau 

99 sigma = self.sigma 

100 

101 lam_high = c / (self.freq_high_factor * self.design_freq) 

102 l_min = 0.5 * lam_high * self.length_factor # shortest (front) element 

103 

104 # k = 0 longest (back) ... k = n-1 shortest (front). 

105 lengths = [l_min / tau ** (n - 1 - k) for k in range(n)] 

106 half = [le / 2 for le in lengths] 

107 # spacing d_k = 2*sigma*l_k between element k and k+1 

108 x = [0.0] 

109 for k in range(n - 1): 

110 x.append(x[-1] + 2 * sigma * lengths[k]) 

111 return half, x 

112 

113 def build_wires(self): 

114 eps = 0.05 

115 wavelength = 299.792458 / self.design_freq 

116 quarter = 0.25 * wavelength 

117 half, x = self._layout() 

118 n = int(self.n_elements) 

119 z = self.base 

120 

121 tups = [] 

122 for k in range(n): 

123 h = half[k] 

124 xk = x[k] 

125 L = (xk, -h, z) 

126 C0 = (xk, -eps, z) 

127 C1 = (xk, eps, z) 

128 R = (xk, h, z) 

129 arm = self.segs_for(h - eps, quarter) 

130 # left arm, named centre gap (a feeder port), right arm 

131 tups.append((L, C0, arm, None, None)) 

132 tups.append((C0, C1, self.segs_for(2 * eps, quarter), None, f"d{k}")) 

133 tups.append((C1, R, arm, None, None)) 

134 return tups 

135 

136 def build_network(self): 

137 half, x = self._layout() 

138 n = int(self.n_elements) 

139 z0 = self.z0 

140 

141 ports = {f"d{k}": PortOnWire(f"d{k}") for k in range(n)} 

142 branches = [] 

143 # Crossed feeder section between adjacent element centres. Length = 

144 # physical boom spacing; transposed=True gives the 180-degree 

145 # section-to-section phase reversal of the transposed feeder. 

146 for k in range(n - 1): 

147 branches.append( 

148 TL( 

149 a=f"d{k}", 

150 b=f"d{k + 1}", 

151 z0=z0, 

152 length=x[k + 1] - x[k], 

153 transposed=True, 

154 ) 

155 ) 

156 # Driven at the front (shortest) element. 

157 return Network( 

158 ports=ports, 

159 branches=branches, 

160 sources=[Driven(port=f"d{n - 1}", voltage=1 + 0j)], 

161 )
