1"""Off-Center-Fed dipole (Windom): a half-wave fed away from the middle 

2(L. B. Cebik, W4RNL). 

3 

4A resonant half-wave wire fed at the centre shows ~70 ohm. As the feedpoint 

5slides toward one end the resistive impedance climbs smoothly -- ~100 ohm part 

6way out, ~200 ohm near the one-third point, ~300 ohm and up further still -- 

7while staying essentially resistive at resonance. The OCF (the modern, 

8two-wire-feed descendant of the 1929 single-wire Windom) exploits that: feeding 

9about a THIRD of the way from one end lands near 200-300 ohm, a 4:1 or 6:1 

10balun step from coax, and -- because the same off-centre point is also near a 

11current loop on several harmonics -- gives a multiband doublet. 

12 

13This fills the "off-centre feed" gap: every other dipole in the catalog 

14(invvee, fan, trap, folded) is centre-fed. The defining, testable physics here 

15is the feedpoint impedance's dependence on feed POSITION, not the (ordinary 

16dipole) pattern. 

17 

18Geometry, in the framework's (x, y, z) convention: 

19 - y : the wire axis; total length ~half-wave, the feed gap offset toward -y 

20 - x, z : the wire sits at x = 0, z = base 

21 - HORIZONTALLY POLARISED, ordinary dipole figure-8 broadside off +/- x. 

22 

23 o------------F=================================o z = base 

24 |<-- ~1/3 -->|<------------- ~2/3 ------------->| 

25""" 

26 

27from ... import AntennaBuilder 

28from types import MappingProxyType 

29 

30 

31class Builder(AntennaBuilder): 

32 default_params = MappingProxyType( 

33 { 

34 "design_freq": 28.57, 

35 "freq": 28.57, 

36 "base": 7.0, 

37 # Total wire length as a fraction of a wavelength (~resonant 

38 # half-wave; slightly short for end effects). 

39 "length_frac": 0.485, 

40 # Feed position as a fraction of the total length measured from the 

41 # near (-y) end. ~0.18 (just under a fifth of the way in) lands near 

42 # the classic ~200 ohm Windom point on a thin free-space wire; the 

43 # textbook "one-third from the end" is nearer 100 ohm here. 

44 "feed_frac": 0.18, 

45 "ui_params": MappingProxyType( 

46 { 

47 # OCF feed is high-ish; reference SWR to a 4:1-balun 200 ohm. 

48 "target_z0": 200.0, 

49 "default_view": "xy", 

50 "length_frac": { 

51 "min": 0.44, 

52 "max": 0.52, 

53 }, 

54 "feed_frac": { 

55 "min": 0.1, 

56 "max": 0.5, 

57 }, 

58 } 

59 ), 

60 } 

61 ) 

62 

63 def build_wires(self): 

64 eps = 0.05 

65 wavelength = 299.792458 / self.design_freq 

66 quarter = 0.25 * wavelength 

67 

68 length = self.length_frac * wavelength 

69 z = self.base 

70 

71 # Wire spans y = -length/2 .. +length/2; feedpoint a fraction 

72 # `feed_frac` of the length in from the -y end. 

73 y_left = -length / 2 

74 y_right = length / 2 

75 y_feed = y_left + self.feed_frac * length 

76 

77 L = (0.0, y_left, z) 

78 F0 = (0.0, y_feed - eps, z) 

79 F1 = (0.0, y_feed + eps, z) 

80 R = (0.0, y_right, z) 

81 

82 left_arm = (y_feed - eps) - y_left 

83 right_arm = y_right - (y_feed + eps) 

84 

85 tups = [] 

86 tups.append( 

87 (L, F0, self.segs_for(left_arm, quarter), None) 

88 ) # short arm (to -y end) 

89 tups.append((F0, F1, 1, 1 + 0j)) # off-centre feed 

90 tups.append( 

91 (F1, R, self.segs_for(right_arm, quarter), None) 

92 ) # long arm (to +y end) 

93 return tups
