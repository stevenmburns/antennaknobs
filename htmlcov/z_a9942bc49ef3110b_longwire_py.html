1"""Resonant multi-wavelength long-wire (L. B. Cebik, W4RNL). 

2 

3A single straight wire several wavelengths long is the canonical "long-wire" 

4antenna. While a half-wave dipole fires broadside, a wire grown past 1 wl no 

5longer does: as it passes 1 wl, 2 wl, 3 wl ... its azimuth pattern breaks into 

6more and more lobes, and those lobes form CONES tilted toward the wire's own 

7axis. The more wavelengths of wire, the more lobes, the more the strongest 

8lobes swing toward the ends, and the higher the gain. This model is a 

9CENTER-FED wire about 3.5 wl long (seven half-waves): center feeding it at an 

10odd number of quarter-waves from each end lands the driving point on a current 

11maximum, keeping the impedance moderate and well-conditioned -- an END feed 

12would sit at a high-impedance voltage antinode (thousands of ohms) instead, 

13and a 3.0 wl center feed would sit on a current null for the same reason. The 

14wire lies horizontal along one axis, so it is HORIZONTALLY POLARISED. 

15 

16Methodology purpose: this is the catalog's longest single open conductor, so 

17it stresses the engines' segmentation of a long, multi-half-wave standing-wave 

18structure and the resulting multi-lobe far field -- a different stress from the 

19compact resonant antennas. It is the simplest standing-wave member of the 

20long-wire family that the catalog's `vbeam` and `rhombic` build on (a V-beam 

21is two such wires at an apex; a rhombic is two V-beams nose to nose). 

22 

23Geometry, in the framework's (x, y, z) convention: 

24 - y : the wire axis; the wire runs from -L/2 to +L/2 along y 

25 - z : constant height `base` (the wire lies horizontal) 

26 - x : the pattern fans out in cones about the y axis; broadside is +/- x 

27The structure is a single straight conductor in x = 0, z = base. 

28 

29 A=========F=========B z = base (single ~3.5 wl wire along y) 

30 | | | 

31 -L/2 centre +L/2 F: one-segment driven gap at the current max 

32""" 

33 

34from ... import AntennaBuilder 

35from types import MappingProxyType 

36 

37 

38class Builder(AntennaBuilder): 

39 default_params = MappingProxyType( 

40 { 

41 "design_freq": 28.57, 

42 "freq": 28.57, 

43 # Height of the horizontal wire above ground. 

44 "base": 7.0, 

45 # Overall wire length as a fraction of a wavelength. ~3.5 wl is 

46 # SEVEN half-waves total -> each half is an odd number of quarter 

47 # waves, so the centre feed lands on a current maximum and the 

48 # driving-point R stays moderate (a 3.0 wl wire, by contrast, is 

49 # six half-waves: the centre is a current null at thousands of 

50 # ohms). 3.48 wl trims the reactance toward resonance. 

51 "length_frac": 3.48, 

52 # Overall scale knob the optimiser tunes for resonance (X -> 0). 

53 "length_factor": 1.0, 

54 "ui_params": MappingProxyType( 

55 { 

56 # Center-fed at a current maximum of a ~3.5 wl wire -> a 

57 # moderate, low-hundreds-of-ohms resistance (~140 ohm). 

58 "target_z0": 140.0, 

59 # Wire runs along y in the plane x=0; the yz view shows it 

60 # face-on (length along y, height along z). 

61 "default_view": "yz", 

62 # Degenerate with length_factor (length = length_frac * wl * 

63 # length_factor); pin it and keep length_factor as the knob. 

64 "length_frac": {"hidden": True}, 

65 "length_factor": { 

66 "min": 0.9, 

67 "max": 1.1, 

68 }, 

69 } 

70 ), 

71 } 

72 ) 

73 

74 def build_wires(self): 

75 eps = 0.05 

76 

77 wavelength = 299.792458 / self.design_freq 

78 quarter = 0.25 * wavelength 

79 

80 length = self.length_frac * wavelength * self.length_factor 

81 half = length / 2 

82 

83 z = self.base 

84 

85 # Single straight wire along y, with a one-segment driven gap at the 

86 # centre (the current maximum). Split into a passive left half, the 

87 # driven gap, and a passive right half (cf. half_square / lazy_h 

88 # centre-feed idiom). 

89 left_end = (0.0, -half, z) 

90 gap_m = (0.0, -eps, z) 

91 gap_p = (0.0, eps, z) 

92 right_end = (0.0, half, z) 

93 

94 tups = [] 

95 # Left half (end -> -eps), passive. 

96 tups.append((left_end, gap_m, self.segs_for(half - eps, quarter), None)) 

97 # Driven feed gap across the centre. 

98 tups.append((gap_m, gap_p, 1, 1 + 0j)) 

99 # Right half (+eps -> end), passive. 

100 tups.append((gap_p, right_end, self.segs_for(half - eps, quarter), None)) 

101 

102 return tups
