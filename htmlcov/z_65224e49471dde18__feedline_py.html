1"""Feedline-network utilities used by multi-band antennas in daisy-chain 

2drive mode. 

3 

4The single function `daisy_chain_z_in` takes the antenna's open-circuit 

5N-port Z matrix and stitches it together with N-1 50Ω transmission-line 

6jumpers, returning the single driving-point impedance the rig sees at 

7the main-feed end. Port 0 is the main-feed connection; ports 1..N-1 

8are subsequent stops along the chain; port N-1 is the open end. 

9 

10Usage from momwire: build Y_sc via `BSplineSolver.compute_y_matrix`, 

11invert to get Z_oc, pass to `daisy_chain_z_in`. PyNEC's TL card path 

12solves the same problem natively (no Python-side math) — this helper 

13is just for the momwire half. 

14""" 

15 

16from __future__ import annotations 

17 

18import numpy as np 

19 

20_C_LIGHT = 299_792_458.0 

21 

22 

23def daisy_chain_z_in( 

24 z_oc: np.ndarray, 

25 jumper_lengths_m: list[float], 

26 freq_mhz: float, 

27 z0_ohms: float = 50.0, 

28) -> complex: 

29 """Driving-point impedance at port 0 with the chain feedline applied. 

30 

31 z_oc: N×N open-circuit impedance matrix of the antenna (V = Z·I). 

32 jumper_lengths_m: length-(N-1) sequence of lossless 50Ω jumpers 

33 connecting port i to port i+1 along the chain. 

34 freq_mhz: operating frequency in MHz. 

35 z0_ohms: characteristic impedance of the jumpers (50 by default). 

36 

37 Returns the complex Z_in seen at port 0 when current is injected 

38 there and port N-1 is left open. 

39 

40 Math: each lossless TL segment of length L contributes nodal- 

41 admittance terms y11 = y22 = -j cot(βL)/Z0 and y12 = y21 = 

42 j csc(βL)/Z0 between the two endpoint nodes. Adding those to 

43 Y_ant = inv(Z_oc) gives the combined nodal Y matrix; Z_in = 

44 inv(Y_combined)[0, 0] when the input current is at node 0 and 

45 all other nodes have no external excitation (open end at N-1 

46 is implicit — its KCL row has zero external current). 

47 """ 

48 n_ports = z_oc.shape[0] 

49 if z_oc.shape != (n_ports, n_ports): 

50 raise ValueError(f"z_oc must be square; got shape {z_oc.shape}") 

51 if len(jumper_lengths_m) != n_ports - 1: 

52 raise ValueError( 

53 f"need {n_ports - 1} jumper lengths for {n_ports} ports, " 

54 f"got {len(jumper_lengths_m)}" 

55 ) 

56 if n_ports == 1: 

57 return complex(z_oc[0, 0]) 

58 

59 omega = 2.0 * np.pi * freq_mhz * 1e6 

60 beta = omega / _C_LIGHT 

61 

62 y = np.linalg.inv(z_oc) 

63 

64 for i, length in enumerate(jumper_lengths_m): 

65 bl = beta * length 

66 sin_bl = np.sin(bl) 

67 if abs(sin_bl) < 1e-12: 

68 # Quarter-wave-multiple — the line shorts/opens through; 

69 # numerically catastrophic and physically degenerate. Nudge 

70 # the length a hair to dodge the singularity. Real antennas 

71 # don't sit exactly here either; this only matters if the 

72 # user dials the slider onto it. 

73 bl += 1e-9 

74 sin_bl = np.sin(bl) 

75 cos_bl = np.cos(bl) 

76 y_self = -1j * cos_bl / (z0_ohms * sin_bl) 

77 y_mut = 1j / (z0_ohms * sin_bl) 

78 y[i, i] += y_self 

79 y[i + 1, i + 1] += y_self 

80 y[i, i + 1] += y_mut 

81 y[i + 1, i] += y_mut 

82 

83 # I_ext = [Iin, 0, ..., 0]; Iin = 1 → V = inv(Y) · I_ext → V[0] = Z_in. 

84 z_combined = np.linalg.inv(y) 

85 return complex(z_combined[0, 0])
