1r"""80 m triangular skyloop run on 17 m, matched to 50 Ω with an L-network — 

2the `Shunt`-branch showcase for issue #65 (Q2, shunt-to-common). 

3 

4Geometry is inherited wholesale from `triangular_skyloop`: the same corner-fed 

5equilateral full-wave loop, flown with the `Drone` and meshed off 

6`nominal_nsegs`. Cut for 80 m (~85 m perimeter, 1 λ at 3.8 MHz) but operated on 

7the 17 m band (18.1 MHz), that perimeter is ~4.7 λ and the corner feed is 

8wildly reactive (~225 − 70j here), nowhere near 50 Ω — the classic "loop + 

9tuner" multiband situation. 

10 

11An **L-match** fixes it with two reactive elements. Because R_load > 50 Ω the 

12network is a shunt across the antenna feed plus a series element to the source: 

13 

14 source ──[ series TwoPort: L ]── feed 

15 │ 

16 [ Shunt: C ]── common 

17 

18The port spec drives a virtual `in` node, runs a series `TwoPort(in→feed)`, and 

19puts a `Shunt` across `feed` to the common reference; the impedance read at `in` 

20is the matched input Z. Stock values land it at ~50 Ω / SWR ≈ 1.0 at 18.1 MHz; 

21retune `series_L_uH` / `shunt_C_pF` for other bands or a different loop. 

22 

23`Shunt` — a lumped R/L/C from a single port to the common reference — is the 

24element issue #65 deferred as Q2. A matching network was the motivating use 

25case; this design is it. A lossless matching network doesn't touch the 

26radiation pattern, so the showcase is the impedance/SWR transform, which the 

27reducer computes exactly on top of the extracted antenna Y. 

28""" 

29

43class Builder(TriangularSkyloop): 

44 default_params = MappingProxyType(

46 **TriangularSkyloop.default_params, 

47 # Loop is cut for 80 m (inherited design_freq) but operated on 17 m. 

48 "freq": 18.1, 

49 # L-match elements, tuned for ~50 Ω at 18.1 MHz on the stock loop. 

50 "series_L_uH": 0.873, # series arm, input → feed (TwoPort) 

51 "shunt_C_pF": 59.57, # shunt arm, across the feed (Shunt) 

52 # Coil quality factor (issue #298): adds R = ωL/Q in series with 

53 # the matching inductor. 0 = ideal coil (the historical behavior); 

54 # real air-wound coils run ~50–400. 

55 "coil_q": 0.0, 

56 "ui_params": MappingProxyType(

58 # Matched to 50 Ω, so the SWR readout shows ~1:1. 

59 "target_z0": 50.0,

67 def build_wires(self): 

68 # Reuse the corner-fed triangular_skyloop geometry verbatim; only rename 

69 # the driven chamfer as the network's "feed" port and clear its inline 

70 # excitation, since the L-match network supplies the source at the 

71 # virtual `in` node instead. 

72 wires = [] 

73 for p0, p1, nseg, ex, *rest in super().build_wires(): 

74 wires.append( 

75 (p0, p1, nseg, None, "feed") if ex is not None else (p0, p1, nseg, ex) 

76 ) 

77 return wires 

78 

79 def build_network(self): 

80 r"""L-match, stamped LITERALLY at whatever the sliders say: the MNA 

81 core (issue #285) handles the degenerate endpoints as physics, not 

82 special cases — a 0 H series arm is an ideal wire (Group-2 short 

83 identifying `in` with `feed`) and a 0 F shunt arm is an open (no 

84 element). With both arms zero the matchbox is fully inert — a 

85 pass-through — and the input impedance is just the bare antenna's 

86 (Z_in = Z_ant). `coil_q > 0` gives the series inductor a finite Q 

87 (R = ωL/Q, issue #298), turning the matchbox lossy the way a real 

88 tuner is; 0 keeps the ideal coil.""" 

89 ql = self.coil_q if self.coil_q > 0 else None 

90 return Network( 

91 ports={"feed": PortOnWire("feed"), "in": PortVirtual("in")}, 

92 branches=[ 

93 TwoPort(a="in", b="feed", l=self.series_L_uH * 1e-6, ql=ql), 

94 Shunt(port="feed", c=self.shunt_C_pF * 1e-12), 

95 ], 

96 sources=[Driven(port="in", voltage=1 + 0j)], 

97 )
