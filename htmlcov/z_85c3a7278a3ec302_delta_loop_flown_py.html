1"""A delta loop flown *entirely* by the Drone, with NO explicit trig in the 

2script -- not even the closed-form apex height, and no ``ry`` reflection. Every 

3leg's length is discovered by flying to a plane, including the top edge, which 

4uses ``forward_to_plane(..., factor=2)`` to cross the symmetry plane ``y = 0`` 

5and land on the mirrored corner in a single command. 

6 

7The flight, given the feed height, the top height and the apex tilt: 

8 

9 S --up the right slant--> A (fly to the plane z = top) 

10 A --across the top------> B (fly to the symmetry plane y = 0, factor=2) 

11 B --down the left slant-> T (fly to the plane z = base) 

12 T --feed gap------------> S (close) 

13 

14The point of this variant is the *tradeoff* it exposes: the construction could 

15hardly be simpler -- no cos/sin/tan anywhere, the drone's matrices and the plane 

16solves carry all of it -- but the **parameterization pays for it**. You give the 

17two heights and the tilt; you do NOT set the loop's size or its resonance 

18directly. The perimeter is whatever falls out of ``(base, top, angle_deg)``, so 

19to tune resonance you nudge ``top`` and re-measure. The shipped ``delta_loop`` 

20hands that knob back by solving the apex height in closed form (explicit trig) so 

21you can size by total wire length outright. Simple to build vs. convenient to 

22design with are different axes -- this loop trades the second for the first. 

23 

24The loop is vertical (planar in x = 0), fed by a short driven gap at the bottom. 

25""" 

26 

27from types import MappingProxyType 

28 

29from ... import AntennaBuilder, Drone 

30 

31 

32class Builder(AntennaBuilder): 

33 default_params = MappingProxyType( 

34 { 

35 "design_freq": 28.47, 

36 "freq": 28.47, 

37 # Feed-gap height (the bottom point of the loop). 

38 "base": 7.0, 

39 # Top-edge height. The vertical extent (top - base) and the tilt set 

40 # the size; the perimeter is emergent, not a knob. 

41 "top": 10.0, 

42 # Slant tilt from horizontal (60 deg -> the classic ~equilateral 

43 # delta loop). Given to the drone as a plain yaw -- no trig here. 

44 "angle_deg": 60.0, 

45 "ui_params": MappingProxyType( 

46 { 

47 "target_z0": 100.0, 

48 "default_view": "yz", # loop lies in the x = 0 plane 

49 "top": {"min": 8.5, "max": 11.5}, 

50 "angle_deg": {"min": 45.0, "max": 75.0}, 

51 } 

52 ), 

53 } 

54 ) 

55 

56 def build_wires(self): 

57 eps = 0.05 

58 wavelength = 299.792458 / self.design_freq 

59 quarter = 0.25 * wavelength 

60 n_body = self.nominal_nsegs 

61 n_feed = max(3, self.nominal_nsegs // 7) 

62 

63 S = (0.0, eps, self.base) # right feed terminal 

64 drone = Drone(position=S, nominal_nsegs=n_body, ref=quarter) 

65 

66 # Nose out along the top edge with world +x as "up" so every yaw stays 

67 # in the loop plane (x = 0), then tilt up onto the right slant. No 

68 # direction cosines by hand -- the drone's rotation holds them. 

69 drone.face(heading=(0.0, 1.0, 0.0), up=(1.0, 0.0, 0.0)) 

70 drone.yaw(self.angle_deg) 

71 

72 drone.pay_out() 

73 drone.forward_to_plane((0.0, 0.0, 1.0, self.top)) # S -> A: up to the top 

74 drone.yaw(180.0 - self.angle_deg) # exterior angle at the top-right corner A 

75 # A -> B: fly across the top to the symmetry plane y = 0 and the same 

76 # distance past it, landing on B without computing the top's width. 

77 drone.forward_to_plane((0.0, 1.0, 0.0, 0.0), factor=2.0) 

78 drone.yaw(180.0 - self.angle_deg) # exterior angle at the top-left corner B 

79 drone.forward_to_plane((0.0, 0.0, 1.0, self.base)) # B -> T: down to the feed 

80 drone.feed(1 + 0j) 

81 drone.close(nsegs=n_feed) # T -> S: the driven feed gap, fly home 

82 

83 return drone.wires()
