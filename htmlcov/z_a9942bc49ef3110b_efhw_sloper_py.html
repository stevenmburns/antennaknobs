1"""End-fed half-wave sloper with a real 49:1 unun — "the POTA antenna, 

2complete" (issue #329). 

3 

4The classic park activation: ~10 m of thin wire hoisted to a mast apex, 

5sloping down to a feed point near the ground, fed through a step-down 

6transformer and a short run of coax. This design composes every station 

7piece the modelling arcs built — `Transformer` with core loss (#301), 

8lossy line (#297), lossy wire (#316–#318) — and the power budget (#299) 

9answers the end-fed question honestly: *where do the watts go in a 49:1?* 

10 

11Physics worth knowing before turning the knobs: 

12 

13* **The end of a half wave is a voltage antinode.** The feed impedance 

14 there is a few kΩ — which is why the unun exists. `unun_ratio` picks 

15 the classic step-downs: 49:1 (7:1 turns, ~2450 Ω → 50 Ω), 64:1 (8:1, 

16 ~3200 Ω), 225:4 (7.5:1, ~2812 Ω). The feed sits near the half-wave 

17 ANTI-resonance, historically a numerically nasty spot — the wire loss 

18 modelled since v0.23 damps that singularity, which is also physically 

19 why a real EFHW is more forgiving than an ideal-wire model suggests. 

20* **The unun is not free.** Its magnetizing branch (`lmag_uH` shunting 

21 the 50 Ω side, with core-loss Q `qlmag`) burns a visible slice of the 

22 power budget — the (mag) row. Real FT240-43-class 49:1s measure 

23 85–90 % efficient; the defaults land in that range. The ~`comp_c_pF` 

24 across the primary is the compensation capacitor every published 

25 build hangs there — it tames the transformer's HF rolloff. 

26* **The counterpoise is a knob, not a footnote.** `cp_len_m` defaults to 

27 ~0.05 λ (the classic minimal counterpoise); the coax shield past the 

28 unun plays this role in many field setups. Shrink it and watch the 

29 feedpoint conditioning and SWR drift. 

30* **The wire is a knob** (`wire_type`, the `WIRES` catalog): 28 AWG PVC 

31 is the POTA classic here, and the high-current half-wave middle makes 

32 gauge loss matter more than on a centre-fed dipole — the *wire loss 

33 (I²R)* budget row and the weight readout quantify the tradeoff. 

34 

35The default `length_factor` is tuned so the stock 28 AWG PVC wire 

36presents its best rig-side match near 14.1 MHz at the default slope over 

37average ground; bare or thicker wire tunes higher — retune with the 

38length knob (the insulated-wire velocity factor, same story as 

39`dipoles.pota_invvee`). 

40 

41Geometry is a `Drone` flight in the x–z plane: fly the counterpoise 

42horizontally (along −x) to the feed point at (0, 0, `h_feed`), pitch up 

43by `slope_deg`, lay the short named "ant" gap wire (ports live in wire 

44interiors — the feed needs its own short wire), then the radiator to the 

45apex, climbing toward −x. The slope is a *rise angle*, not an apex 

46height, so every knob combination is a valid sloper — the apex simply 

47lands at h_feed + length·sin(slope_deg) (≈10 m at the defaults, a 

48typical mast). A sloper fires mostly downhill, off the low feed end — 

49the radiator climbs toward −x precisely so the main lobe lands on **+x** 

50(the workbench's forward direction). 

51 

52 apex (derived) 

53 / 

54 / radiator ≈ λ/2 · length_factor (wire_type from WIRES) 

55 / slope_deg above horizontal 

56 F========= z = h_feed main lobe → +x 

57 counterpoise (+x) F = "ant" port → unun (49:1) → coax → rig 

58"""

138 # 40 m: the same POTA box — unun, comp cap, coax untouched — with twice 

139 # the wire. The ~19 m radiator needs a gentler rise: 30° lands the apex 

140 # near 11 m (a tall mast or a friendly tree limb), and the counterpoise 

141 # scales to ~0.05 λ. length_factor retuned for the stock 28 AWG PVC wire: 

142 # rig-side SWR 1.36 at 7.1 MHz. The 100 pF comp cap is a 20 m-flavored 

143 # compromise that caps the match here (~200 pF would buy 1.19 — turn the 

144 # knob if you'd rebuild the unun), and the longer thin wire burns ~10 % 

145 # in I²R (efficiency 84 %) — the gauge tradeoff, amplified. At ~0.26 λ 

146 # up the pattern is near-NVIS (takeoff ≈ 78°, essentially omni) — which 

147 # is exactly how 40 m POTA actually works: regional skywave, not DX.

158 def build_wires(self): 

159 eps = 0.05 

160 wavelength = 299.792458 / self.design_freq 

161 quarter = 0.25 * wavelength 

162 length = 0.5 * wavelength * self.length_factor 

163 

164 # Fly it: face −x so the radiator climbs toward −x and the sloper's 

165 # downhill main lobe lands on +x; counterpoise in to the feed 

166 # point, pitch up by the rise angle (Drone pitch is nose-down 

167 # positive), gap wire, radiator. 

168 d = Drone( 

169 (self.cp_len_m, 0.0, self.h_feed), 

170 nominal_nsegs=self.nominal_nsegs, 

171 ref=quarter, 

172 ) 

173 d.face((-1.0, 0.0, 0.0)) 

174 d.pay_out().forward(self.cp_len_m) 

175 d.pitch(-self.slope_deg) 

176 d.forward(eps, nsegs=1) 

177 d.forward(length - eps) 

178 wires = d.wires() 

179 # The short gap edge becomes the named "ant" port wire: the port 

180 # interrupts the current path between counterpoise and radiator — 

181 # the end-fed's feed. 

182 p0, p1, nsegs, _ev = wires[1] 

183 wires[1] = (p0, p1, nsegs, None, "ant") 

184 return wires 

185 

186 def build_network(self): 

187 turns = UNUN_TURNS[self.unun_ratio] 

188 branches = [ 

189 # Step-down unun: rig side "pri" sees Z_feed / turns². 

190 Transformer( 

191 a="pri", 

192 b="ant", 

193 n=1.0 / turns, 

194 lmag=self.lmag_uH * 1e-6, 

195 qlmag=self.qlmag if self.qlmag > 0 else None, 

196 ), 

197 ] 

198 if self.comp_c_pF > 0: 

199 branches.append(Shunt(port="pri", c=self.comp_c_pF * 1e-12)) 

200 branches.append(TL.from_cable(self.cable, "rig", "pri", self.line_len_m)) 

201 return Network( 

202 ports={ 

203 "ant": PortOnWire("ant"), 

204 "pri": PortVirtual("pri"), 

205 "rig": PortVirtual("rig"), 

206 }, 

207 branches=branches, 

208 sources=[Driven(port="rig", voltage=1 + 0j)], 

209 )
