1"""OWA Yagi: 4 elements, whole-band flat 50-ohm feed (NW3Z/WA3FET concept, 

2systematized by L. B. Cebik, W4RNL). 

3 

4The Optimized Wideband Antenna is a Yagi with one deliberately "wasted" 

5element: a FIRST DIRECTOR parked unusually close to the driver (~0.05 wl 

6here, vs ~0.18 wl to the next director). That element barely adds gain -- 

7it is a COUPLED RESONATOR that flattens the driving-point impedance at 

850 ohm across an entire band: the matching network IS an element, so the 

9feed is direct coax with no beta/gamma/stub hardware. Cebik published OWA 

10families for 20/10/6/2 m; this is his 10 m 4-element on a 13.5' boom 

11(reflector 17.68', driver 17.20' @ 5.12', D1 16.14' @ 6.87', D2 15.20' 

12@ 13.0', 1" tubing), with lengths held as wavelength fractions so the 

13design scales with `design_freq`. 

14 

15What the model reproduces (free space, length_factor 0.985 to centre the 

16band for this segmentation): SWR(50) stays 1.10-1.32 over ALL of 

1728.0-29.0 MHz while gain runs ~8.3-8.7 dBi and F/B ~12-14 dB. The two 

18contrasts that make the point: 

19 - the catalog's `beams.yagi` (a conventional driver-reflector-directors 

20 design) swings past 5:1 SWR over the same span; 

21 - delete D1 (or drag `d1_length_factor` off unity) and the OWA's own 

22 match collapses to 2.7-5.9 -- the wideband feed lives in that one 

23 close-coupled element, not in the driver. 

24The trade Cebik documents: ~0.2 dB of peak gain given up for total-band 

25consistency, and fat elements matter (thin-wire OWAs narrow; the 1" tubing 

26is part of the published design, modelled via the wire-material spec). 

27 

28Geometry, in the framework's (x, y, z) convention: 

29 - y : element axis (all four elements parallel to y) 

30 - x : boom / firing axis; the beam fires +x (toward the directors) 

31 - z : constant height `base` 

32 

33 refl driver D1 D2 

34 | | | | 

35 | F | | (F = direct 50-ohm feed) 

36 | | | | 

37 x= 0 .148 .198 .375 (wl) 

38 ^^ the OWA coupled resonator 

39""" 

40 

41from antennaknobs import AntennaBuilder 

42from antennaknobs.network import WireSpec 

43from types import MappingProxyType 

44 

45 

46class Builder(AntennaBuilder): 

47 # Cebik's 10 m 4-el OWA (ant36), feet -> fractions of the 28.4 MHz 

48 # wavelength: (element half-length, boom position). 

49 TABLE = ( 

50 (0.255250, 0.0), # reflector 

51 (0.248319, 0.147837), # driver 

52 (0.233017, 0.198367), # D1 -- the coupled resonator 

53 (0.219445, 0.375366), # D2 

54 ) 

55 DRIVER = 1 

56 D1 = 2 

57 

58 default_params = MappingProxyType( 

59 { 

60 "design_freq": 28.4, 

61 "freq": 28.4, 

62 "base": 10.0, 

63 # Overall element-length scale. 0.985 centres the published 

64 # design's flat-SWR window on 28.0-29.0 MHz for this 

65 # segmentation (Cebik's feet assume his NEC model's meshing). 

66 "length_factor": 0.985, 

67 # Scale on D1 alone -- the knob that shows the OWA mechanism: 

68 # off unity, the whole-band match collapses while the pattern 

69 # barely moves. 

70 "d1_length_factor": 1.0, 

71 "ui_params": MappingProxyType( 

72 { 

73 "target_z0": 50.0, 

74 # Single-band 10 m beam: snap the GUI sweep to the band. 

75 "sweep_policy": {"band_locked": True}, 

76 "default_view": "xy", 

77 "length_factor": { 

78 "min": 0.95, 

79 "max": 1.02, 

80 }, 

81 "d1_length_factor": { 

82 "min": 0.9, 

83 "max": 1.1, 

84 }, 

85 } 

86 ), 

87 } 

88 ) 

89 

90 def build_wire_material(self): 

91 # 1" aluminum tubing idealized as PEC, held as a wavelength fraction 

92 # (0.0127 m at 28.4 MHz) so the fat-element behaviour -- which the 

93 # OWA's bandwidth depends on -- survives rescaling to other bands. 

94 wavelength = 299.792458 / self.design_freq 

95 return WireSpec(radius=0.001203 * wavelength) 

96 

97 def build_wires(self): 

98 eps = 0.05 

99 wavelength = 299.792458 / self.design_freq 

100 quarter = 0.25 * wavelength 

101 b = self.base 

102 

103 tups = [] 

104 for i, (half_frac, pos_frac) in enumerate(self.TABLE): 

105 half = half_frac * wavelength * self.length_factor 

106 if i == self.D1: 

107 half *= self.d1_length_factor 

108 x = pos_frac * wavelength 

109 if i == self.DRIVER: 

110 # Driver: one-segment centre gap carries the direct feed. 

111 arm = self.segs_for(half - eps, quarter) 

112 tups.append(((x, -half, b), (x, -eps, b), arm, None)) 

113 tups.append( 

114 ((x, -eps, b), (x, eps, b), self.segs_for(2 * eps, quarter), 1 + 0j) 

115 ) 

116 tups.append(((x, eps, b), (x, half, b), arm, None)) 

117 else: 

118 ns = self.segs_for(2 * half, quarter) 

119 tups.append(((x, -half, b), (x, half, b), ns, None)) 

120 return tups
