1"""Horizontal full-wave loop / "loop skywire" (L. B. Cebik, W4RNL). 

2 

3A single closed loop whose perimeter is about one wavelength, lying FLAT in a 

4plane of constant height (z = base) and fed at the midpoint of one side. The 

5horizontal full-wave loop is the classic all-band "loop skywire": strung low 

6over real ground it is a strong NVIS radiator (it fires nearly straight up, the 

7near-vertical-incidence skywave that fills the close-in skip zone), while in 

8free space it is HORIZONTALLY POLARISED with its main lobe broadside to the 

9loop plane -- i.e. straight up and straight down, along +/- z. A full-wave loop 

10presents a moderate, near-resistive feedpoint of roughly 100-130 ohm. We use a 

11SQUARE loop (side = perimeter / 4, ~0.25-0.27 wl per side); Cebik's max-pattern 

12full-wave loop runs the perimeter a few percent over 1.0 wl, so the side is a 

13touch over a quarter wave. 

14 

15Methodology purpose: this is a LARGE single closed loop with ONE feed in a NEW 

16orientation (horizontal, in a constant-z plane). It exercises the engines' 

17closed-loop Y assembly and the overhead-hemisphere (zenith-pointing) far field. 

18Single-port closed loops are already supported by every engine, so this serves 

19partly as a control case and partly as a new-orientation / large-loop check. 

20 

21Geometry, in the framework's (x, y, z) convention: 

22 - z : height; the whole loop sits in the plane z = base (it is HORIZONTAL) 

23 - x, y : the square's four corners at (+/- h, +/- h, base) 

24 - main lobe along +/- z (broadside to the loop plane -> toward zenith) 

25 

26 D-----------C y = +h 

27 | | 

28 | | (loop lies flat at z = base; viewed from above) 

29 F | feed F at the midpoint of side A->D 

30 | | 

31 A-----------B y = -h 

32 x=-h x=+h 

33""" 

34 

35from antennaknobs import AntennaBuilder 

36from types import MappingProxyType 

37 

38 

39class Builder(AntennaBuilder): 

40 default_params = MappingProxyType( 

41 { 

42 "design_freq": 28.57, 

43 "freq": 28.57, 

44 # Height of the (flat) loop plane above ground. In free space this 

45 # only shifts the structure; over real ground it sets the NVIS 

46 # takeoff. Low for a skywire. 

47 "base": 5.0, 

48 # Side length as a fraction of a wavelength. Four sides -> a ~1 wl 

49 # perimeter; ~0.264 wl per side runs the perimeter a few percent 

50 # over 1.0 wl (Cebik's max-pattern full-wave-loop proportion) and 

51 # is tuned so the free-space feed is near resonant (X ~ 0) at 

52 # length_factor = 1. 

53 "side_frac": 0.2635, 

54 # Overall scale knob the optimiser tunes for resonance (X -> 0). 

55 # A full-wave loop is naturally a bit inductive; this trims it. 

56 "length_factor": 1.0, 

57 "ui_params": MappingProxyType( 

58 { 

59 # Full-wave loop -> moderate, near-resistive feed ~100 ohm. 

60 "target_z0": 100.0, 

61 # The loop lies flat in z = base; its x and y spans are the 

62 # two largest, so the xy view shows the loop face-on. 

63 "default_view": "xy", 

64 # Degenerate with length_factor (side = side_frac * wl * 

65 # length_factor); pin it and keep length_factor as the knob. 

66 "side_frac": {"hidden": True}, 

67 "length_factor": { 

68 "min": 0.9, 

69 "max": 1.1, 

70 }, 

71 } 

72 ), 

73 } 

74 ) 

75 

76 def build_wires(self): 

77 eps = 0.05 

78 

79 wavelength = 299.792458 / self.design_freq 

80 quarter = 0.25 * wavelength 

81 

82 side = self.side_frac * wavelength * self.length_factor 

83 h = side / 2.0 

84 z = self.base 

85 

86 # Keep the per-segment length roughly uniform across wires (good NEC 

87 # practice at the corner junctions). Odd counts so the side centre 

88 # falls on a segment. 

89 # Four corners of the flat square, all at z = base. 

90 A = (-h, -h, z) 

91 B = (h, -h, z) 

92 C = (h, h, z) 

93 D = (-h, h, z) 

94 

95 # Feed gap at the midpoint of side A->D (the wire runs along +y here). 

96 feed = 2 * eps 

97 F0 = (-h, -feed / 2.0, z) # gap start, just below the midpoint 

98 F1 = (-h, feed / 2.0, z) # gap end, just above the midpoint 

99 

100 tups = [] 

101 # Side A->D split into: A->gap-start (passive), gap (driven, 1 seg), 

102 # gap-end->D (passive). The remaining three sides close the loop. 

103 tups.append((A, F0, self.segs_for(h - feed / 2.0, quarter), None)) 

104 tups.append((F0, F1, 1, 1 + 0j)) # one-segment driven gap 

105 tups.append((F1, D, self.segs_for(h - feed / 2.0, quarter), None)) 

106 tups.append((D, C, self.segs_for(side, quarter), None)) # top side 

107 tups.append((C, B, self.segs_for(side, quarter), None)) # right side 

108 tups.append( 

109 (B, A, self.segs_for(side, quarter), None) 

110 ) # bottom side, closes the loop 

111 return tups
