1"""Rectangle "magnetic slot" SCV: a flattened 1 wl loop (L. B. Cebik, W4RNL). 

2 

3From Cebik's "SCVs: A Family Album" -- the same self-contained-vertical 

4family as the catalog's `half_square`, `bobtail`, `bruce`, and 

5`right_angle_delta`: 1 wl wire shapes that park their current maxima in 

6vertical wire sections, so the verticals' fields add (vertically polarised, 

7low takeoff, no radials) while the horizontal members largely cancel. 

8 

9This member is the 1 wl loop flattened into a wide, short rectangle -- 

10Cebik's 40 m numbers are 56' x 12.8' (~0.41 x 0.09 wl), fed at the CENTRE of 

11one short vertical side. The squashed proportions put both short sides' 

12current maxima to work and buy gain: 4.4 dBi free space, second only to the 

13half-square's 4.6 in the family album. This model reads ~5.1 vs its own 

14half-square's 4.9 -- it runs the whole family a few tenths hot and flips 

15Cebik's 0.2 dB ordering, but the robust claim survives: the two are the 

16family's top pair, over a dB clear of the delta. The "magnetic slot" name is 

17the marketing physics Cebik was debunking -- it behaves as a plain loop SCV, 

18no exotic mode. 

19 

20The price is the feedpoint: flattening the loop drops the mid-side feed to 

21~15 ohm (Cebik's figure; this wire reads ~13 at length_factor 1.0325). Cebik 

22suggests voltage feed via a tapped parallel-tuned circuit; the 

23transmission-line fix modelled here is a QUARTER-WAVE TRANSFORMER of low-Z 

24line -- two paralleled 50 ohm coaxes make z0 ~ 25 ohm, and 25^2/13.1 lands 

25~48 ohm at the shack (this model: SWR 1.05, dead resonant). The 

26instructive contrast with `wire.edz`: the EDZ's series line works by 

27ROTATION around the SWR circle to its low-R crossing; here the feed is 

28already near-resonant, so the quarter-wave section is the classic R-SCALING 

29transformer (z_shack = z0^2 / z_feed) instead. 

30 

31Extensions Cebik also modelled, worth knowing: the Double Magnetic Slot 

32(paralleled loops with a Mobius crossing; 4.7 dBi, a friendlier 80 ohm) and 

33the end-fed Open Double Slot (110.8' x 11.1'; 5.7 dBi, 30 ohm). 

34 

35Geometry, in the framework's (x, y, z) convention: 

36 - y : the long horizontal wires run along y 

37 - z : height; bottom wire at `base`, top wire ~0.09 wl higher 

38 - x : broadside; bidirectional off +/- x with end-on rejection 

39The structure is planar in x = 0. The matching section is an electrical 

40element (a `TL` branch), not geometry. 

41 

42 D===================================C z = base + v (top, ~0.41 wl) 

43 F | 

44 | | F = feed, centre of the left 

45 A===================================B z = base short side 

46 : 

47 : quarter-wave low-Z section (TL branch, z0_match) 

48 S S = shack feed (virtual port) 

49""" 

50 

51from antennaknobs import AntennaBuilder 

52from antennaknobs.network import Driven, Network, PortOnWire, PortVirtual, TL 

53from types import MappingProxyType 

54 

55 

56class Builder(AntennaBuilder): 

57 default_params = MappingProxyType( 

58 { 

59 "design_freq": 28.57, 

60 "freq": 28.57, 

61 # Height of the bottom wire above ground. Low, like the other 

62 # SCVs: the short vertical sides do the work. 

63 "base": 3.0, 

64 # Long (horizontal) side as a fraction of a wavelength; Cebik's 

65 # 56' on 40 m. 

66 "horiz_frac": 0.407, 

67 # Short (vertical) side as a fraction of a wavelength; Cebik's 

68 # 12.8' -- the flattening that trades feed R for gain. 

69 "vert_frac": 0.093, 

70 # Quarter-wave transformer: characteristic impedance (ohm) and 

71 # physical length as a fraction of a wavelength. ~25 ohm is two 

72 # paralleled 50 ohm coaxes; sqrt(14 * 50) ~ 26 would be exact. 

73 "z0_match": 25.0, 

74 "match_len_frac": 0.25, 

75 # Overall scale knob; 1.0325 is resonance (X ~ 0 at the bare 

76 # feed) for this segmentation. 

77 "length_factor": 1.0325, 

78 "ui_params": MappingProxyType( 

79 { 

80 "target_z0": 50.0, 

81 "default_view": "yz", 

82 "length_factor": { 

83 "min": 0.9, 

84 "max": 1.1, 

85 }, 

86 "z0_match": { 

87 "min": 10.0, 

88 "max": 75.0, 

89 "step": 0.5, 

90 "precision": 1, 

91 }, 

92 "match_len_frac": { 

93 "min": 0.05, 

94 "max": 0.45, 

95 }, 

96 } 

97 ), 

98 } 

99 ) 

100 

101 def build_wires(self): 

102 wavelength = 299.792458 / self.design_freq 

103 quarter = 0.25 * wavelength 

104 lf = self.length_factor 

105 

106 w = self.horiz_frac * wavelength * lf 

107 v = self.vert_frac * wavelength * lf 

108 half_w = w / 2 

109 zb = self.base 

110 zt = zb + v 

111 

112 A = (0.0, -half_w, zb) # bottom-left 

113 B = (0.0, half_w, zb) # bottom-right 

114 C = (0.0, half_w, zt) # top-right 

115 D = (0.0, -half_w, zt) # top-left 

116 

117 # Feed gap centred on the LEFT short side -- the current maximum. 

118 pe = 0.1 # feed-edge length, m 

119 F0 = (0.0, -half_w, zb + (v - pe) / 2) 

120 F1 = (0.0, -half_w, zb + (v + pe) / 2) 

121 

122 leg = self.segs_for((v - pe) / 2, quarter) 

123 return [ 

124 # Left short side: bottom corner -> feed edge -> top corner. 

125 (A, F0, leg, None, None), 

126 (F0, F1, 1, None, "feed"), 

127 (F1, D, leg, None, None), 

128 # Top, right side, and bottom close the loop. 

129 (D, C, self.segs_for(w, quarter), None, None), 

130 (C, B, self.segs_for(v, quarter), None, None), 

131 (B, A, self.segs_for(w, quarter), None, None), 

132 ] 

133 

134 def build_network(self): 

135 wavelength = 299.792458 / self.design_freq 

136 return Network( 

137 ports={ 

138 "feed": PortOnWire("feed"), # centre of the left short side 

139 "shack": PortVirtual("shack"), # bottom of the transformer 

140 }, 

141 branches=[ 

142 TL( 

143 a="shack", 

144 b="feed", 

145 z0=self.z0_match, 

146 length=self.match_len_frac * wavelength, 

147 ), 

148 ], 

149 sources=[Driven(port="shack", voltage=1 + 0j)], 

150 )
