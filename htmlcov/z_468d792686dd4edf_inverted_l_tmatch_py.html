1r"""10 m inverted-L worked on the 12 m band through a T-network tuner — 

2the first design with a pure interior circuit node (series C, shunt L, 

3series C), exercising the MNA network core on the classic "wire antenna + 

4T-match" situation. 

5 

6Geometry is inherited wholesale from `inverted_l`: the same 10 m 

7(28.57 MHz) bent, top-loaded vertical with four elevated radials. Operated 

8on 24.9 MHz that riser is electrically short and the feed is the textbook 

9tuner case — Z_ant ≈ 10.8 − 121.7j Ω (low R, big capacitive X), nowhere 

10near 50 Ω. 

11 

12A **T-network** (the standard ham "T-tuner" topology, a high-pass tee) 

13fixes it with two series capacitors flanking a shunt inductor: 

14 

15 source ──[ C1 series ]──┬──[ C2 series ]── feed 

16 │ 

17 [ L shunt ]── common 

18 

19The port spec drives a virtual `in` node, chains `TwoPort` capacitors 

20through a virtual midpoint `m`, and hangs a `Shunt` inductor off `m`. The 

21midpoint is a pure interior circuit node — no antenna segment, no TL, 

22nothing but lumped Group-2 branches meeting at a KCL row — which no other 

23design exercises. Stock values land ~50 Ω / SWR ≈ 1.0005 at 24.9 MHz. 

24 

25Because the antenna is short (R ≈ 11 Ω) the match must ride a virtual 

26resistance of ~2 kΩ, so there is no symmetric-capacitor solution and the 

27loaded Q is high (~13) — the real-world "narrow retune on a short 

28antenna" behavior, visible here as a sharp SWR notch in a frequency sweep. 

29 

30Degenerate slider endpoints (all legal since the MNA core, issue #285): 

31a 0 H shunt inductor hard-shorts the midpoint to common (input becomes 

32pure C1 reactance); a 0 F series capacitor is an OPEN — note this is the 

33opposite convention from the L-match's arms, where zero meant "element 

34absent": an absent series capacitor in a T-network is C → ∞ (a wire), not 

35C → 0. Setting `series_c2_pF = 0` disconnects the antenna entirely (the 

36source then sees only C1 + L), and `series_c1_pF = 0` open-circuits the 

37source itself. 

38"""

85 def build_wires(self): 

86 # Reuse the inverted-L geometry verbatim; rename the driven base gap 

87 # as the network's "feed" port and clear its inline excitation — the 

88 # T-network supplies the source at the virtual `in` node instead. 

89 wires = [] 

90 for p0, p1, nseg, ev, *rest in super().build_wires(): 

91 if ev is not None: 

92 wires.append((p0, p1, nseg, None, "feed")) 

93 else: 

94 wires.append((p0, p1, nseg, ev, *rest)) 

95 return wires

97 def build_network(self): 

98 return Network( 

99 ports={ 

100 "feed": PortOnWire("feed"), 

101 "m": PortVirtual("m"), 

102 "in": PortVirtual("in"), 

103 }, 

104 branches=[ 

105 TwoPort(a="in", b="m", c=self.series_c1_pF * 1e-12), 

106 Shunt( 

107 port="m", 

108 l=self.shunt_l_uH * 1e-6, 

109 ql=self.coil_q if self.coil_q > 0 else None, 

110 ), 

111 TwoPort(a="m", b="feed", c=self.series_c2_pF * 1e-12), 

112 ], 

113 sources=[Driven(port="in", voltage=1 + 0j)], 

114 )
