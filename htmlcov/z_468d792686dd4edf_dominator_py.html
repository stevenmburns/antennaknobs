1"""KJ6ER's "Dominator" — end-fed halfwave vertical with 49:1 transformer. 

2 

3Third of the KJ6ER trio (see verticals.pota_performer and 

4verticals.challenger). The full halfwave stands on end: a 25' telescoping 

5whip is the radiator, fed at its bottom end through a 49:1 (or 56:1) 

6step-down transformer ~4' up a tripod, with a long ~33% λ linked 

7counterpoise off the transformer ground sloping to the earth and along 

8it. The vertical EFHW — lowest takeoff of the trio. Plans (rev 2025-02): 

9https://www.vhfclub.org/pdf/Dominator%20Antenna%20by%20KJ6ER%20(2025-02).pdf 

10 

11Published claims this model duplicates (4NEC2, 15M = 21.350 MHz, 

12"Computer Model" lengths 272" radiator / 175" counterpoise): 

13peak +0.60 dBi @ 18° elevation, −3 dB beamwidth 27°, SWR 1.006 (at 

142450 Ω); "structural efficiency" 99.5%; measured transformer insertion 

15loss −0.96 dB (TennTennas 49:1) or −0.40 dB (MyAntennas 56:1, the 

16"Dominator+" — the `plus` variant). Note the honest asymmetry KJ6ER 

17himself points out: the EFHW's transformer is the lossiest component in 

18any of his three antennas — ~20% of input power for the stock 49:1 — 

19which is the same story our wire.efhw_sloper budget tells. 

20 

21As with the Challenger, `lmag_uH`/`qlmag` are calibrated so the budget's 

22(mag) row reproduces the measured insertion loss at 21.35 MHz, not 

23derived from core data. The end-fed's high-Z feed uses the same 

24gap-wire + counterpoise conditioning as wire.efhw_sloper. 

25"""

106 def build_wire_material(self): 

107 return _ALUMINUM 

108 

109 def build_wires(self): 

110 eps = 0.05 

111 h = self.h_feed 

112 

113 droop = math.asin(min(1.0, max(0.0, h - self.cp_end_h) / self.cp_len_m)) 

114 

115 return [ 

116 ((0, 0, h), (0, 0, h + eps), 1, None, "ant"), 

117 ((0, 0, h + eps), (0, 0, h + self.whip_len_m), self.nominal_nsegs, None), 

118 ( 

119 (0, 0, h), 

120 ( 

121 self.cp_len_m * math.cos(droop), 

122 0.0, 

123 h - self.cp_len_m * math.sin(droop), 

124 ), 

125 max(7, self.nominal_nsegs // 2), 

126 None, 

127 ), 

128 ] 

129 

130 def build_network(self): 

131 turns = XFMR_TURNS[self.xfmr_ratio] 

132 return Network( 

133 ports={"ant": PortOnWire("ant"), "rig": PortVirtual("rig")}, 

134 branches=[ 

135 Transformer( 

136 a="rig", 

137 b="ant", 

138 n=1.0 / turns, 

139 lmag=self.lmag_uH * 1e-6, 

140 qlmag=self.qlmag if self.qlmag > 0 else None, 

141 ) 

142 ], 

143 sources=[Driven(port="rig", voltage=1 + 0j)], 

144 )
