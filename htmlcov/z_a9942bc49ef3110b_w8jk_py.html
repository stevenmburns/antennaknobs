1"""W8JK flat-top beam: a 2-element all-driven 180-degree array (L. B. Cebik, 

2W4RNL, after John Kraus W8JK). 

3 

4Two horizontal wires run parallel, close-spaced (~1/8 wavelength), and are fed 

5180 degrees OUT OF PHASE -- in the real antenna by a parallel feeder with one 

6side given a half-twist between the two element centres. The out-of-phase, 

7close-spaced pair radiates BIDIRECTIONALLY endfire (off the +/- x ends of the 

8boom, in the plane of the wires) with a deep broadside null. Using "extended" 

9elements longer than a half-wave (~0.64 wl each, the Kraus extended-double-Zepp 

10length) puts the collinear gain on top of the array gain, so the flat-top beats 

11a plain dipole by several dB while staying dead simple and feedline-tunable. 

12 

13This fills the "out-of-phase driven array" gap: the catalog's lazy_h feeds two 

14elements IN phase (broadside), the hb9cv feeds them ~135 degrees apart 

15(unidirectional endfire); the W8JK is the 180-degree, bidirectional member of 

16the family. Because the two elements are mirror images across the y-z plane and 

17are driven with equal-and-opposite voltages, the antisymmetric excitation 

18forces equal-magnitude, opposite-phase currents -- exactly the 180-degree 

19phasing the half-twist feeder produces -- so we model it as two centre feeds at 

20+1 and -1 rather than as an explicit crossed line. 

21 

22Cebik / Kraus proportions: elements 0.5-1.25 wl long (0.64 wl "extended" here), 

23spacing 1/8 to 1/2 wl (0.125 wl here). HORIZONTALLY POLARISED. 

24 

25Geometry, in the framework's (x, y, z) convention: 

26 - y : the element length axis (both wires run along y) 

27 - x : boom/firing axis; the two wires sit at x = -spacing/2 and +spacing/2, 

28 and the beam fires bidirectionally off +/- x 

29 - z : constant height `base` 

30 

31 x = -s/2 ====================F==================== (rear, V = -1) 

32 x = +s/2 ====================F==================== (front, V = +1) 

33 beam <--> +/- x (broadside null off +/- y) 

34""" 

35 

36from antennaknobs import AntennaBuilder 

37from types import MappingProxyType 

38 

39 

40class Builder(AntennaBuilder): 

41 default_params = MappingProxyType( 

42 { 

43 "design_freq": 28.57, 

44 "freq": 28.57, 

45 "base": 7.0, 

46 # Element length as a fraction of a wavelength. ~0.64 wl is the 

47 # Kraus "extended" length that adds collinear gain. 

48 "elem_frac": 0.64, 

49 # Element-to-element spacing as a fraction of a wavelength (~1/8). 

50 "spacing_frac": 0.125, 

51 "ui_params": MappingProxyType( 

52 { 

53 # Out-of-phase close-spaced feed -> low, reactive driving 

54 # point; in practice tuned via the parallel feeder. Use a 

55 # representative open-wire reference. 

56 "target_z0": 100.0, 

57 # Wires span y, boom along x -> the xy view is face-on. 

58 "default_view": "xy", 

59 "elem_frac": { 

60 "min": 0.5, 

61 "max": 1.25, 

62 }, 

63 "spacing_frac": { 

64 "min": 0.1, 

65 "max": 0.5, 

66 }, 

67 } 

68 ), 

69 } 

70 ) 

71 

72 def build_wires(self): 

73 eps = 0.05 

74 wavelength = 299.792458 / self.design_freq 

75 quarter = 0.25 * wavelength 

76 

77 elem = self.elem_frac * wavelength 

78 spacing = self.spacing_frac * wavelength 

79 half = elem / 2 

80 z = self.base 

81 

82 def element(x, voltage): 

83 """A horizontal wire along y at (x, z), centre-fed with `voltage`.""" 

84 L = (x, -half, z) 

85 R = (x, half, z) 

86 C0 = (x, -eps, z) 

87 C1 = (x, eps, z) 

88 return [ 

89 (L, C0, self.segs_for(half - eps, quarter), None), 

90 (C0, C1, self.segs_for(2 * eps, quarter), voltage), 

91 (C1, R, self.segs_for(half - eps, quarter), None), 

92 ] 

93 

94 tups = [] 

95 tups.extend(element(-spacing / 2, -1 + 0j)) # rear, anti-phase 

96 tups.extend(element(spacing / 2, 1 + 0j)) # front 

97 return tups
