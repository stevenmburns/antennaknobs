1"""End-fed half-wave "Zepp" with a tuned-stub feeder (L. B. Cebik, W4RNL). 

2 

3The Zeppelin antenna is a half-wave radiator fed at ONE END through a section 

4of open-wire "tuned feeder". The end of a half-wave sits at a current MINIMUM / 

5voltage MAXIMUM, so the driving-point impedance there is very HIGH (thousands of 

6ohms) and strongly reactive; the quarter-wave-ish stub transforms that down to 

7something a rig can drive. The radiator is HORIZONTALLY POLARISED with the usual 

8half-wave dipole pattern (the feeder is meant to be non-radiating). 

9 

10This is the IDEAL-TRANSMISSION-LINE cousin of the catalog's `jpole`, and the 

11contrast is the point. `jpole` builds the matching stub as PHYSICAL parallel 

12wires (real geometry, modelled by the MoM solver), whereas here the feeder is an 

13IDEAL series `TL` network branch from the antenna's end port to a virtual shack 

14port. That makes the Zepp a deliberate METHODOLOGY PROBE with two findings: 

15 

16 1. Feeding the network layer at a near-OPEN antenna port (a current null) is 

17 ILL-CONDITIONED: the bare end impedance varies ~2x between solver bases 

18 (e.g. ~2900 -19900j ohm on NEC2/sinusoidal vs ~1400 -13700j on the rooftop 

19 bases) -- the same current-null hazard as `bobtail`'s and `bruce`'s feeds. 

20 The series stub transforms that spread along with the impedance: the shack 

21 R agrees across bases but the shack REACTANCE inherits the spread. 

22 2. A LOSSLESS line preserves the reflection magnitude, and an end-fed half 

23 wave is nearly total reflection, so NO single series feeder length can 

24 bring the shack impedance near 50 ohm -- it only rotates around the rim of 

25 the Smith chart (R stays a few ohms, X swings through +/- a kilohm). The 

26 historical Zepp ran its tuned feeders to an antenna TUNER for exactly this 

27 reason; a coax-direct match needs a PARALLEL shorted stub, which is what 

28 `jpole` realises in physical wire. We model the honest tuner-fed feeder. 

29 

30As always the radiator's GAIN and PATTERN are basis-stable (~2.1 dBi dipole); 

31the transformed feed impedance is indicative and tuner-fed, not a 50 ohm match. 

32 

33Geometry, in the framework's (x, y, z) convention: 

34 - y : the half-wave radiator axis (end-fed at y = 0, open at y = +half) 

35 - z : constant height `base` 

36 - x : broadside (figure-8 dipole pattern) 

37The tuned feeder is an electrical element (a `TL` branch), not geometry. 

38 

39 F=========================================> z = base (half-wave radiator) 

40 | (end feed, high-Z voltage antinode) 

41 | tuned stub (TL branch, z0_stub) 

42 S S = shack feed (virtual port) 

43""" 

44 

45from antennaknobs import AntennaBuilder 

46from antennaknobs.network import Driven, Network, PortOnWire, PortVirtual, TL 

47from types import MappingProxyType 

48 

49 

50class Builder(AntennaBuilder): 

51 default_params = MappingProxyType( 

52 { 

53 "design_freq": 28.57, 

54 "freq": 28.57, 

55 "base": 10.0, 

56 # Radiator length as a fraction of a wavelength (~half wave). 

57 "length_frac": 0.48, 

58 # Tuned-stub feeder: characteristic impedance (ohm) and physical 

59 # length as a fraction of a wavelength. A ~quarter-wave high-Z stub 

60 # transforms the very high end impedance down toward the rig. 

61 "z0_stub": 600.0, 

62 "stub_len_frac": 0.25, 

63 "length_factor": 1.0, 

64 "ui_params": MappingProxyType( 

65 { 

66 "target_z0": 50.0, 

67 "default_view": "yz", 

68 # Degenerate with length_frac (length = length_frac * wl * 

69 # length_factor); pin length_factor and keep length_frac, the 

70 # curated knob. 

71 "length_factor": {"hidden": True}, 

72 "length_frac": { 

73 "min": 0.40, 

74 "max": 0.55, 

75 }, 

76 "z0_stub": { 

77 "min": 300.0, 

78 "max": 800.0, 

79 "step": 1.0, 

80 "precision": 1, 

81 }, 

82 "stub_len_frac": { 

83 "min": 0.1, 

84 "max": 0.45, 

85 }, 

86 } 

87 ), 

88 } 

89 ) 

90 

91 def build_wires(self): 

92 eps = 0.05 

93 wavelength = 299.792458 / self.design_freq 

94 quarter = 0.25 * wavelength 

95 

96 length = self.length_frac * wavelength * self.length_factor 

97 z = self.base 

98 

99 # End feed: a one-segment named gap "ant" at y=0 (the open end / voltage 

100 # antinode), then the half-wave radiator running out to y=+length. 

101 # No direct voltage source -- the tuned stub drives this port. 

102 return [ 

103 ((0.0, 0.0, z), (0.0, eps, z), 1, None, "ant"), 

104 ( 

105 (0.0, eps, z), 

106 (0.0, length, z), 

107 self.segs_for(length - eps, quarter), 

108 None, 

109 ), 

110 ] 

111 

112 def build_network(self): 

113 wavelength = 299.792458 / self.design_freq 

114 stub_len = self.stub_len_frac * wavelength 

115 return Network( 

116 ports={ 

117 "ant": PortOnWire("ant"), # high-Z end of the radiator 

118 "shack": PortVirtual("shack"), # bottom of the tuned stub 

119 }, 

120 branches=[ 

121 TL(a="shack", b="ant", z0=self.z0_stub, length=stub_len), 

122 ], 

123 sources=[Driven(port="shack", voltage=1 + 0j)], 

124 )
